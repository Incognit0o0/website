# Инструкция по локальному запуску проекта

Этот проект представляет собой Full-stack приложение на React (Vite) и Express с использованием PostgreSQL.

## 1. Требования
Убедитесь, что на вашем компьютере установлены:
- **Node.js** (версия 18 или выше)
- **PostgreSQL** (если хотите использовать постоянную базу данных)
- **npm** (идет в комплекте с Node.js)

## 2. Установка
1. Скачайте архив с кодом из AI Studio (меню Settings -> Export to ZIP) и распакуйте его.
2. Откройте терминал в папке проекта.
3. Установите зависимости:
   ```bash
   npm install
   ```

## 3. Настройка окружения
1. Создайте файл `.env` в корневой директории (можно скопировать `.env.example`).
2. Добавьте в него строку подключения к вашей базе данных:
   ```env
   DATABASE_URL="postgresql://postgres:password@localhost:5432/horserace"
   ```
   *Если переменная `DATABASE_URL` не будет задана, сервер автоматически перейдет в режим работы "в памяти" (данные будут сбрасываться при перезагрузке).*

## 4. Схема базы данных
Если вы хотите настроить таблицы вручную, используйте файл:
- `schema.sql` — содержит структуру таблиц `rooms` и `history`.

## 5. Запуск
Для запуска в режиме разработки выполните:
```bash
npm run dev
```
После запуска сайт будет доступен по адресу: `http://localhost:3000`

## 5. Сборка для продакшена (для деплоя на VPS)
1. Соберите фронтенд:
   ```bash
   npm run build
   ```
2. Запустите сервер:
   ```bash
   npm run start
   ```

## Структура проекта
- `server.ts` — Основной файл сервера (Express + WebSockets + PostgreSQL).
- `src/` — Исходный код фронтенда (React).
- `dist/` — Папка со скомпилированным фронтендом для деплоя.
